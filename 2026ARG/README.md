# Brutalist Countdown Website

A minimalist, institutional-style single-page website with a full-screen background video and countdown timer.

## Setup Instructions

1. **Add your video file**: Place your video file in the root directory and name it `your-video.mp4`

2. **Update the Google Form URL**: In `index.html`, replace `[YOUR_GOOGLE_FORM_URL]` with your actual Google Form URL

3. **Add the meta description content**: In `index.html`, replace `[PASTE THE 'YOU NEED TO BE...' LIST HERE]` with your actual list content

4. **Open the website**: Simply open `index.html` in a web browser

## Features

- Full-screen grayscale background video (30% opacity)
- Countdown timer to January 24, 2026, 00:00:00 EST
- Brutalist design: black background, dim grey text, monospace font
- Automatic status change and link hiding when timer reaches zero

## Browser Compatibility

Works best in modern browsers that support:
- HTML5 video
- CSS3 filters (grayscale)
- ES6 JavaScript
